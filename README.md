# Zbloue

AI-powered code analysis for beginner developers — friendly, explainable, and shareable.

Zbloue helps beginner developers understand, improve, and share code in a more human-friendly way. It turns raw code into clear explanations, actionable fixes, and shareable outputs designed for learning.

## Why Zbloue?

Many beginner developers and students struggle to understand code, find bugs, and learn from dry or overly technical analysis.

Zbloue is built to make code analysis easier to understand, more educational, and more shareable.

## Features

- Beginner-friendly code explanations
- Three analysis modes
- Clear summaries and actionable fixes
- Line-by-line insights
- Bug detection and improvement suggestions
- Shareable card output
- Public snippet page
- No signup required for the MVP

## How It Works

1. Paste or write code in the editor
2. Choose an analysis mode
3. Get a friendly explanation, fixes, and suggestions
4. Share the result as a card or snippet page

## What Makes It Different

- Built for learning, not just reviewing
- Friendly and explainable output
- Shareable results for developers and students
- Designed for beginner to intermediate users

## Tech Stack

- Next.js
- TypeScript
- Supabase
- AI-powered analysis pipeline
